JekOS Installer




Provide your AWS credentials to provision a new EC2 instance in your AWS account and install the JekOS kernel there. We recommend using a user specifically created for JekOS. Make sure that this key/user has privileges necessary to run EC2 instances, for example:.

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Action": "ec2:*",
      "Effect": "Allow",
      "Resource": "*"
    },
  ]
}
